BFC-块级格式化上下文

2016-07-23
块级格式化上下文 是web页面可视html渲染的一部分。它是块盒子的布局发生及浮动体彼此交互的区域。

简单来说，BFC能创建一个内部与外界互不影响的盒子。在实际项目中，盒子外边距折叠、包容盒子坍塌之类的情况可以用BFC原理解决。

块级格式化上下文由以下之一创建：
  • 根元素或其它包含它的元素
  • 浮动 (元素的 float 不是 none)
  • 绝对定位的元素 (元素具有 position 为 absolute 或 fixed)
  • 内联块 inline-blocks (元素具有 display: inline-block)
  • 表格单元格 (元素具有 display: table-cell，HTML表格单元格默认属性)
  • 表格标题 (元素具有 display: table-caption, HTML表格标题默认属性)
  • 块元素 元素具有overflow 值不是 visible
  • display: flow-root

前端缓存概要

2016-07-22

一、http的缓存

对于h5应用来说，频繁地与服务器进行大量数据交互是很慢的。重复利用缓存是一个性能优化的重要手段。
浏览器自带缓存的功能，但是需要我们正确设置http头才能达到目的。

缓存

缓存是指存储指定资源的一份拷贝，并在下次请求该资源时提供该拷贝的技术。当 web 缓存发现请求的资源已经被存储，它会拦截请求，返回该资源的拷贝，而不会去源服务器重新下载。

前端知识碎片

2016-03-15

执行上下文

首先明确几个概念：

  • EC：函数执行环境（或执行上下文），Execution Context
  • ECS：执行环境栈，Execution Context Stack
  • VO：变量对象，Variable Object
  • AO：活动对象，Active Object
  • scope chain：作用域链

每次当控制器转到可执行代码的时候，就会进入一个执行上下文。执行上下文可以理解为当前代码的执行环境，它会形成一个作用域。JavaScript中的运行环境大概包括三种情况。

  • 全局环境：JavaScript代码运行起来会首先进入该环境
  • 函数环境：当函数被调用执行时，会进入当前函数中执行代码
  • eval（不建议使用，可忽略）
用async优雅的实现异步
undefined和null区别

一个重要的区别，服务于不同的目的和理由。区分这两个值，你可以认为undefined代表一个意想不到的没有值而null作为预期没有值的代表。typeof undefined为字符串“undefined”，typeof null为“Object”

原型

每个函数都有一个原型属性prototype指向函数的自身原型，而每个有这个函数创建的对象都有一个proto指向自己的原型，然后逐层深入直到最后Object的原型对象null

跨域解决方案

2016-02-21

一、跨域资源共享（CORS）
在服务器设置接受前端跨域请求， Access-Control-Allow-Origin: *，前端浏览器支持跨域资源共享，双方符合即可进行跨域请求。
二、JSONP
Web 页面上调用 js 文件不受浏览器同源策略的影响，所以通过 Script 便签可以进行跨域的请求（但是无法处理POST请求，而且无法判断请求的失败与否。。。只能通过超时时间来确定）

零与零之间

2013-03-28


　　　零与零之间（BETWEEN ZERO & ZERO）
　　　零
　　　站在沙滩，望眼过去只是一条水天相接的白线，这种无穷无尽的感觉倒不是因为距离有多远，而是弯曲的海面阻断了视线，看不到前方。人生其实也并不长，只是未来的不确定让人有了苍茫的视线，望不到未来，便感觉人生很长。五年前的我对人生就是这样看待，所以并未料想到今天望着沙滩会如此的感慨。一波又一波的浪，气势汹汹地翻滚而来，蔓延到脚下又退去，接着下一波又来，又退去。这很像一波波新生又退去的人潮，像人生当中许多从零到零的片段。远处的海面与脚下的浪花一样平静、温情，这样看来便觉得中间涌起的浪花纵使震撼美丽，看看岸边的平静也觉得这浪花很是矫情、多余，找不到意义。是啊，不只是那么一刹那让我感觉中间的惊涛骇浪是多余的。每一波奔腾而来的浪像演绎着一场人生，平静的海平面与岸边就像生命的生与死。每个人的一生或青春或童年都是从零开始而后又归零，每一波浪潮都是从平静中掀起而后又归于沙滩的平静。至今，我还对生命中存在如此多的故事——在零与零之间循环的故事，而感到不解和恐惧。
　　　所以每当站在沙滩回望过去的情景时，便会这样强烈地想着每个故事，想象着这些故事是如何变成我的过去、我的未来、我的整个生命。我参与着许多人的生命，许多人又组成了我的生命。我们都在演绎着一段又一段的故事，从零开始回归到零，每个生命都是如此；循环往复，永不改变，这是上帝对人类安排的最神奇的一笔。
